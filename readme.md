# Magnification

## Setup

### Using UV

```bash
uv venv
uv pip install -r requirements.txt
uv run app.py
```

### Using Conda

```bash
conda create --name magnify python=3.11 -y
conda activate magnify
pip install -r requirements.txt
python app.py
```

### Standard Python
```bash
pip install -r requirements.txt
flask run app.py
```
